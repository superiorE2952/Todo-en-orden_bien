# Titulo: Todo en orden?

Este proyecto implementa y compara diferentes algoritmos de ordenamiento utilizando Python.  
Incluye visualización en consola y análisis de complejidad mediante gráficas.

---

# Funcionalidades

- Implementación de algoritmos de ordenamiento:
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort

- Visualización de los algoritmos mediante histogramas en consola
- Comparación de tiempos de ejecución
- Gráfica de complejidad (O(n²) vs O(n log n))
